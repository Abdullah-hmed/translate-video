# Setup

1. Clone the respository

```bash 
git clone https://github.com/Abdullah-hmed/translate-video.git
```

2. download the ([mel-band-roformer](https://github.com/KimberleyJensen/Mel-Band-Roformer-Vocal-Model.git)) git repository within the repository
```bash
git submodule add https://github.com/KimberleyJensen/Mel-Band-Roformer-Vocal-Model.git mel_band_roformer
```
